# Array functions
This is an example exercise shows different usage of array based functions in es6 way.

## Prerequisites
<ul>
  <li><a href="https://nodejs.org">NodeJs</a></li>
  <li><a href="http://babeljs.io">Babel</a></li>
  <li><a href="https://webpack.js.org">Webpack</a></li>
</ul>

## Steps
```
// Installation of packages.json
$ npm install
```
```
// Run your own server
$ npm run start // http://localhost:3000
```
```
// create build folder
$ npm run build
```
